# Marketing Assets

Archivio ufficiale degli asset creativi KhamaKey.

Questa cartella e' condivisa tra agenti: Antigravity genera, Codex/Cursor normalizzano, il team approva e pubblica.

## Struttura

```text
marketing/
├── asset-manifest.md
├── landing/
├── marketplace/
├── product/
├── reseller/
├── social/
├── video/
└── source-antigravity/
```

## Regole

- `landing/`: hero, sezioni visuali e demo statiche per pagine vendita.
- `marketplace/`: immagini e copy visuale per Amazon, TikTok Shop, Shopify listing.
- `product/`: immagini prodotto, mockup marketplace, hero Shopify.
- `reseller/`: materiale per rivenditori, WhatsApp, brochure, listini visuali.
- `social/`: post Instagram/Facebook/TikTok statici.
- `video/`: script, storyboard, shot list, thumbnail.
- `source-antigravity/`: output sorgente o documenti originali generati da Antigravity.
- Ogni asset pronto deve essere registrato in `asset-manifest.md`.
- Non committare file sopra 5 MB: registrarli nel manifest con posizione esterna o R2.
- Non usare link `file:///Users/...` nei documenti ufficiali.
