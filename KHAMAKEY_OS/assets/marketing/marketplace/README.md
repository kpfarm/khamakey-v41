# Marketplace Marketing Assets

Asset per Shopify, Amazon, TikTok Shop e altri canali di vendita.

## Regole

- Salvare immagini scheda prodotto, infografiche, banner e varianti marketplace.
- Indicare sempre canale, prodotto/SKU e stato nel manifest.
- Non sovrascrivere versioni approvate: creare `v2`, `v3`, ecc.
