# Social Marketing Assets

Post statici, caroselli, thumbnail e creativita' per Instagram, Facebook, TikTok e ads.

## Regole

- Indicare formato nel nome file quando serve: `1x1`, `4x5`, `9x16`.
- Ogni asset usato in ads o pubblicazione organica deve essere nel manifest.
- Gli script video vanno in `../video/`, non in questa cartella.
